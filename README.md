# ProteinLab — Flask versiyonu

Protein sentezi konu anlatımı + 5 mini oyun.

## Kurulum

```bash
pip install -r requirements.txt
python app.py
```

Tarayıcıda: http://localhost:5000

## Sayfalar
- `/` Ana sayfa
- `/anlatim` 10 adımlı konu anlatımı
- `/amino-asitler` 20 amino asit tablosu
- `/oyun` Mini oyunlar (kodon, transkripsiyon, replikasyon, antikodon, adım sıralama)
